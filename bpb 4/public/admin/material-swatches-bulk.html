Upload swatches for every product, all at once

Drag as many product folders as you have (selected all together in Finder, dropped as one batch) and this tool auto-matches every image to its catalog variant. Each product folder becomes its own card — preview all, then hit Upload all once to write the lot.

How this works:
  1. In Finder, select multiple product folders at once (e.g. the Dimensions folder, the Catalina Grana folder, and the Belair Wall folder all selected)
  2. Drag them all into the zone below
  3. Tool creates one card per folder, each detecting its own product family from the folder name
  4. Review each family's preview grid, optionally override if auto-detection is wrong
  5. Click the global Upload all at the top — processes every card in sequence
  6. Variants that already have a swatch are skipped automatically
