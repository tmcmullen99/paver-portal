JobNimbus API probe

Diagnostic console for the JobNimbus API. Master-only. Calls https://app.jobnimbus.com/api1/<endpoint> via the /api/jobnimbus-probe Cloudflare Function. API key is cached in sessionStorage and cleared when the tab closes. Start with the contacts preset to confirm your key works, then try product endpoints.

Request

From JobNimbus → Settings → API → New API Key (Admin access profile). Cached in sessionStorage only; not sent anywhere except to /api/jobnimbus-probe on each probe.
Path after /api1/. No leading slash needed. Query strings OK.
Records per page. Added as ?size=N if not already in the endpoint.

Response

Field inventory

Flattened to 2 levels. Shows how often each field appears across all returned records and one real sample value. Use this to decide which fields to map into catalog_staging.
